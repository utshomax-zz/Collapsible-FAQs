# Collapsible-FAQs
This project is a part of NITR-HarmonyOS assessment.

Simple FAQ webpage with vanilla JavaScript.

No additional library used.

Features
- Collapsible FAQ.
- No dependency.
- Option to mark a FAQ as usefull.
- Sorting based on usefull FAQs.
